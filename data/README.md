# Thư mục Data

Thư mục này chứa các file template và dữ liệu từ vựng.

## Files

- `vocabulary-template.csv` - Template CSV để import từ vựng
- `vocabulary-template.json` - Template JSON để import từ vựng

## Cách sử dụng

1. Copy một trong các template
2. Điền từ vựng của bạn vào
3. Chạy lệnh import:
   ```bash
   npm run import:csv data/your-file.csv
   # hoặc
   npm run import:json data/your-file.json
   ```

Xem thêm hướng dẫn chi tiết trong `README-IMPORT.md`
